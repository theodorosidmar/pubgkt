# PUBG Kotlin Multiplatform library
Use PUBG's official API with your favorite stack. For further information on PUBG's official API, check out their [documentation](https://developer.pubg.com/).

This is a side project and I check it on some times. Feel free to contribute. I appreciate your help. Raise an issue or contact me.

## Status
- JVM ✅
- Node.js - Under construction 👷
- Browser - Under construction 👷
- iOS - Under construction 👷
- Android - Under construction 👷

## Getting started
<details>
    <summary>JVM</summary>

### Maven
```xml
<dependency>
  <groupId>pubgkt</groupId>
  <artifactId>pubgkt-jvm</artifactId>
  <version>0.0.1</version>
</dependency>
```

### Gradle
```
# build.gradle.kts (Kotlin)
dependencies {
    implementation("pubgkt:pubgkt-jvm:0.0.1")
}

# build.gradle (Groovy)
dependencies {
    implementation("pubgkt:pubgkt-jvm:0.0.1")
}
```
</details>
